Borders
use border styles for images, buttons, blocks. You can add border radius, color, position and width
1. Border color

Primary

use for active tabs and not reviewed stories

HTML: <span class="bc-primary"></span>

Secondary

used for blocks that display the time of last visit

HTML: <span class="bc-lighten-1"></span>

Secondary 2

used for reviewed stories

HTML: <span class="bc-lighten-2"></span>

Secondary 3

used for delimit blocks in the menu

HTML: <span class="bc-lighten-3"></span>

2. Border radius

Circle

used for avatars, buttons, notifications

HTML: <span class="br-circle"></span>

XXS

used for cards, hovers for advertising, buttons

HTML: <span class="br-xxs"></span>

XS

used for photo for advertising, hovers for contacts

HTML: <span class="br-xs"></span>

SM

used for stories

HTML: <span class="br-sm"></span>

MD

used for blocks that display the time of last visit

HTML: <span class="br-md"></span>

LG

used for inputs of comment

HTML: <span class="br-lg"></span>

XL

used for inputs of posts

HTML: <span class="br-xl"></span>

XXL

used for input of search

HTML: <span class="br-xxl"></span>

XXXL

used for notifications

HTML: <span class="br-xxxl"></span>
