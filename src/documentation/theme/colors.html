How to use

About

Set the background of an element to any contextual class.

How it works

            @each $key, $value in $primary { 
              .bg-primary-#{$key} { 
                  background-color: $value !important;
                }
            }

Where $key - name of the class, $value - data, that describes property (in our case it's hex-color) and $primary - name of array.

Examples

          <div class="bg-primary">.bg-primary</div>
          <div class="bg-primary-lighten-1">.bg-primary-lighten-1</div>

          <div class="bg-secondary">.bg-secondary</div>
          <div class="bg-secondary-lighten-1">.bg-secondary-lighten-1</div>
