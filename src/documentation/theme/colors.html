How to use

About

Set the background of an element to any contextual class.

How it works

@each $key, $value in $primary {
.bgc-primary-#{$key} {
background-color: $value !important; }
}

Where $key - name of the class, $value - data, that describes property (in our case it's hex-color) and $primary - name of array.

Examples

<p class="bgc-primary">.bgc-primary</p>

<p class="bgc-secondary-lighten-1">.bgc-secondary-lighten-1</p>

<p class="bgc-alert">.bgc-alert</p>
