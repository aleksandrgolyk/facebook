Padding

Config:

-contains coefficients(coef.) that show multiplication by base-rhythm.

$padding: (... "coef.": coef. * $base-rhythm ...);

Helpers:

The classes are named using the format {property}{sides}-{size} for xs and {property}{sides}-{size}-{breakpoint} for sm, md, lg, and xl.

property is padding.

sides is:

  • t - top
  • b - bottom
  • r - right
  • l - left
  • x - horizontal
  • y - vertical
  • blank - for classes that set a padding on all 4 sides of the element

sizes is:

  • 0 - for classes that eliminate the padding by setting it to 0
  • 1 - for classes that set the padding to $base-rhythm * 1
  • 2 - for classes that set the padding to $base-rhythm * 2
  • 3 - for classes that set the padding to $base-rhythm * 3
  • 4 - for classes that set the padding to $base-rhythm * 4
  • 5 - for classes that set the padding to $base-rhythm * 5
  • 6 - for classes that set the padding to $base-rhythm * 6
