Flex

Config:

- Contains lists:

  • flex-wrap - list of flex-wrap property values;
  • flex-grow - list of flex-grow property values;
  • flex-dir - list of flex-direction property values;

- Contains maps:

  • flex-order - map of flex-order property values;
  • justify-content - map of justify-content property values;
  • align-content - map of align-content property values;
  • align-items - map of align-items property values;

Helpers:

The classes are named using the format {list or map name}{key from map or value from list} for xs and {list or map name}{key from map or value from list}-{breakpoint} for sm, md, lg, and xl.

list values:

flex-wrap values

  • wrap
  • wrap-reverse
  • nowrap

flex-grow values

  • 1
  • 2
  • 3

flex-dir values

  • row
  • column

map values:

flex-order key: values

  • first: -1
  • 0: 0
  • 1: 1
  • 2: 2
  • 3: 3
  • last: 4

justify-content key: value

  • start: flex-start
  • end: flex-end
  • center: center
  • between: space-between
  • around: space-around
  • evenly: space-evenly

align-content key: value

  • start: flex-start
  • end: flex-end
  • center: center
  • between: space-between
  • around: space-around
  • stretch: stretch

align-items key: value

  • start: flex-start
  • end: flex-end
  • center: center
  • baseline: baseline
  • stretch: stretch
