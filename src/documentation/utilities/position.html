Position

If you need to place block relative to parent element,
assign it absolute position and relative position for parent element

Config:

-contains map variable with position values.

$padding: ( "abs": absolute, "rel": relative );

Helpers:

There are 2 types of helpers.

  • General.

    Naming format .pos-{type}
    type is a key in position config, short name of position value.

  • Based on screen size.

    Naming format .pos-{type}-{point}.
    point is a short name for media size (xs, sm, md, lg, xl, xxl)
